# Football Stat Site
By: John Winborne

## Overview
A full-stack NFL player stats website built with React and Express. Users can browse players, search and filter by team/position/age, view player details, and save favorites.

## Features
- Browse NFL players
- Search by player name
- Filter by team, position, and age
- View player details and season stats
- Save favorites with localStorage
- Responsive layout for smaller screens

## Tech Stack
### Frontend
- React
- React Router
- Font Awesome
- CSS

### Backend
- Node.js
- Express
- Axios
- CORS
- dotenv

## APIs Used
- SportsData.io / DiscoveryLab NFL API
- TheSportsDB API

## Environment Variables
### Server
- DISCOVERYLAB_API_KEY
- SPORTSDB_API_KEY
- PORT

### Client
- REACT_APP_API_BASE_URL

## Running Locally
start server by:
node server.js 
start client by:
npm start
